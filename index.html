Project 01

01 十字路口红绿灯控制

01
S7-1200 SCL IO 分配

设计十字路口信号灯时序逻辑，包含主干道与支干道红绿灯协调控制、行人过街请求处理、紧急车辆优先通行等功能。完成完整的 IO 分配表，使用 SCL 语言编写控制程序，实现精准的时序切换与故障安全保护。

对称模式 · 东西等时

东西方向与南北方向绿灯时长相同，适用于车流量均衡的十字路口场景。

非对称模式 · 主干优先

主干道绿灯时长大于支干道，配合行人过街请求与车辆感应信号，实现动态配时控制。

Project 02

02 Factory IO 换向传送带

02
Factory IO PLC 联动 换向逻辑

在 Factory IO 仿真环境中搭建传送带 3D 场景，实现物料换向控制。通过 PLC 与 Factory IO 实时联动，编写换向逻辑程序，实现传送带的启动、停止、正反转切换及传感器信号反馈。

Project 03

03 自动化小车（5种模式）

03
S7-1200 LAD SCL 多模式控制

实现自动化小车的五种运动控制模式：手动模式、自动模式、呼叫模式、寸动模式、随动模式。分别使用 LAD（梯形图）和 SCL（结构化文本）两种编程语言实现，对比不同编程范式的实现思路与代码结构。涵盖限位保护、急停处理、模式互锁等安全逻辑。

5种模式切换演示

展示小车在手动、自动、呼叫、寸动、随动五种模式下的完整切换过程与运动表现。

手自动控制 — LAD 梯形图版

使用梯形图（LAD）语言实现的手自动切换控制程序，展示梯形图逻辑的直观性与可靠性。

手自动控制 — SCL 结构化文本版

使用结构化文本（SCL）语言实现的手自动切换控制程序，对比高级语言编程的灵活性与可维护性。

寸动模式专项演示

展示小车在寸动模式下的点动控制与精确定位操作，适用于调试与微调场景。
